查詢班表(Query)
Schedule Qeury
班表查詢
•點選此功能，系統預設會顯示自己的最新班表，若欲查詢他人班表，可選擇年月及員工號查詢
•不輸入員工號，僅選擇月份，則可查詢自己該月份的班表
•班表顯示兩種格式：Landscape（橫式月曆樣式）或Portrait（直式）
•直式班表中的Crew一欄，點選可看同Flight的其他組員名單，並可寄送EMail
•系統僅提供查詢當月及下月班表
Schedule Compare
比對班表
•輸入年月及被比對者的員工號，可比對自己與他人之班表
Flight Query
航班查詢
•以年月日查詢，若不輸入航班編號Flight Number，可查詢該日期所有航班組員資訊
•以年月日查詢，輸入航班編號（Flight Numbe），可查詢航班所有組員資訊
•若輸入航班完整資料（包含年月日及Flight Number與DPT），則直接顯示該航班之組員資訊
•日期預設為不選擇（下拉選單顯示為 N），可查詢該月份所有航班組員資訊
•點選查詢結果中Email欄位的圖示，可線上寄送EMail給該組員
•系統僅提供查詢當月及次月的航班
Flying Time Query
飛時查詢
•以年月日查詢該日期所有航班之飛時，或者加上Flight Number，查詢該班次飛時
Off Crew Query
Off 組員查詢
•以年月日及職務類別(occu)查詢該日Day Off組員

換班專區(Transfer)
Put Schedule
欲換班表

•選定年月，查詢自己的班表，結果以Trip number顯示
•點選Detail欄位的 圖示，可顯示此Trip詳細的Flight Schedule
•若要丟出該筆班表，勾選Put欄位 及 ，可丟出班表至「欲換班表」
•此功能設定之功用，在提供他人查詢自己欲換之班表（即在『查詢可換班表』中，可查詢他人在『欲換班表』中所丟出的航班，以作為換班之參考）
•欲取消所丟出之班表，請至「我的丟班資訊」中設定
Put Schedule Query
查詢可換班表
•以年月查詢所有組員所丟出之班表，查詢結果以Trip Number顯示
•點選Schedule Detail圖示，可顯示此Trip詳細的Flight Schedule
•此功能所顯示之班表，即為其他組員在「欲換班表」中所設定丟出之班表
•查詢結果中，點選姓名欄位可查詢該組員之個人資料
Transfer Record
我的丟班資訊
•可查詢或取消自己在「欲換班表」中丟出的班表紀錄
•丟班記錄中，點選Fltno，可查詢喜好此航班者的資料
•點選Empno，可與該組員比對班表
Favorite Flight Query
喜好航班查詢
•可查詢航班的喜好者統計
•點選Fltno，可查詢喜好該航班者之資料
Make Application
填申請單
•選定年月及被換者員工號，填寫換班申請單
•勾選欲換的航班，或選擇兩人全月互換，由系統計算飛時等數字並驗證。
CheckApplication
申請單記錄
•可依申請年月查詢自己申請，或他人申請而自己為被換者之申請單記錄
Download File
下載選班資訊
•可依選擇年月，下載申換班限額表

個人資料(Personal)
Crew Informatiion
組員個人資料
•可查閱個人基本資料，以＊標註的欄位（如行動電話、住家電話、即時通訊）可以更改
Change Password
變更系統密碼
•可變更密碼
•密碼最少需六位數英.數字
Favorite Flight
自訂最愛航班
•可設定個人最愛航班
•他人丟出班表後，在丟班記錄中，點選該筆資料的Fltno，可查詢喜好此航班者
Friend List
自訂好友名單
•可輸入員工號，自訂好友名單，並可直接查詢好友的班表、個人資料、及寄送Email
•可查詢將自己設為好友之使用者資料，亦可寄送Email
•設為好友或被設為好友者，若選擇不開放班表，則僅會顯示其姓名員工號，但仍不可查詢其詳細個人資料及班表
AL Offsheet
年假輸入/查詢
•可查詢個人年假Quota、請休假資料查詢、特休假申請
Flypay List
飛加清單
•可選擇年月，系統會自動寄送本人的飛加清單至其全員信箱。
Crew Log List
飛航記錄
•可選擇年月，系統會自動寄送本人的飛航記錄至其全員信箱。
Set Schedule Status
鎖定/開放班表
•可自行設定是否鎖定或開放班表
•欲使用查詢他人班表或換班等進階功能，需選擇開放個人班表

其他功能(Others) 
Query Crew & mail Schedule
查詢組員及寄送班表
•以期別或員工號、序號、或姓名為條件查詢其他組員資料
•系統查詢優先順序為姓名、員工號或序號、及期別查詢
•期別請由小至大輸入
•若需查詢單一期別，僅輸入第一欄位即可
•於查詢結果的畫面中，點選Email欄位的圖示，可線上寄送EMail給該組員
•點選最新班表的圖示，可查詢該組員最新班表
•勾選Mail Sche及 ，可將自己的班表寄送至該組員的全員電子信箱
•所寄送之班表為當時正確的班表，但班表有可能因為換班因素有所更動，欲查詢最新班表，請至查詢班表區查詢。
Send Mail
發信給組員
•可線上寄送EMAIL給其他組員
Contact EF
意見信箱
•對系統若有任何意見，可寄送意見至EF或各組
•與本系統無關之事項，請勿使用此發信功能
Online User List
線上使用者
•可觀看系統目前線上使用者列表
•點選Empno欄位，可觀看該使用者的個人資料，但需該使用者設定狀態為「開放班表」，方可查詢。（若使用者設定為「鎖定班表」，則他人無法查詢其資料。）
